# 💰 Expense Tracker

A beginner-friendly expense tracker built with plain HTML, CSS, and JavaScript — no frameworks, no installs needed.

## ✨ Features

- Add income and expense transactions
- Delete any transaction
- Live summary dashboard (balance, total income, total expenses)
- Savings ratio progress bar
- Filter by All / Income / Expenses
- Data saved in browser (localStorage) — survives page refresh

## 🚀 How to Run

1. Download or clone this repository
2. Open `expense-tracker.html` in any web browser
3. That's it — no setup, no installs!

```bash
git clone https://github.com/rojinakarki19/Expense-Tracker.git
cd Expense-Tracker
# Open expense-tracker.html in your browser
```
